## Code Push 기능 도입 고려
ℹ️ Code Push란? 각 플랫폼의 앱스토어 리뷰 과정 없이 앱의 업데이트를 반영할 수 있는 서비스입니다. 앱의 네이티브 코드가 아닌, Dart 코드 및 정적 리소스만의 변경 사항을 빠르게 배포할 수 있습니다.

- 앱의 버그를 수정할때 앱의 심사과정 없이 사용자가 바로 최신 버전을 사용할 수 있게 합니다.
- 사용자가 앱을 다시 다운로드하거나 업데이트하지 않아도 변경사항을 실시간으로 받을 수 있습니다.
- 결제나 기타 중요한 사항에서 발생하는 버그를 신속히 대응함으로써, 빠르게 문제를 해결하고 손실을 최소화할 수 있습니다.
